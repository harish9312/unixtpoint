How to open the Console/Terminal...??

1>> Open the Dash by clicking the Ubuntu icon in the upper-left, type "terminal", and select the Terminal application from the results that appear.

2>> Hit the keyboard shortcut Ctrl - Alt + T .

How to switch DIRECTORIES...??

To Switch Between Directories You can use these commands---
1. To go back to the previous dir.. " cd .. " command is used

2. To Go back to the home directory " cd " command is used

3. To go to the ROOT directory " cd/ " command is used

How To Make Directory..??

To Create New Directories You can use the command---
1. To create an empty dir.. " mkdir " command is used
